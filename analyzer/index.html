Behind the Numbers

How the AI Prop Analyzer Works

Propeller starts with the listed line, evaluates the sport-specific signals that are available, and summarizes the result as a directional confidence score. The exact inputs vary by sport and data availability.

1. Read the Exact Line

PLAYER · STAT · LINE

The analysis is attached to a specific player, stat type, and line. A different line can produce a different research conclusion, so the displayed number always matters.

2. Evaluate Available Signals

SPORT-SPECIFIC

Depending on the sport and market, signals can include role, injuries, matchup, recent form, game environment, historical performance, and market context.

3. Combine the Evidence

AVAILABLE DATA ONLY

Propeller combines the signals that return usable evidence. Missing or unavailable inputs are a limitation to disclose, not a reason to manufacture a neutral reading.

4. Show Direction + Confidence

OVER · UNDER · NEUTRAL

The output summarizes directional conviction for the displayed line. It is a research signal, not a guarantee, calibrated probability, recommendation, or promise of profit.

Read the displayed OVER or UNDER direction first, then the 50–100 model-confidence score. Values near 50 are closer to neutral; higher values show stronger support for the displayed side. The score is not a win probability or guarantee. Read the full methodology for signal definitions, missing-data behavior, and current limitations.
