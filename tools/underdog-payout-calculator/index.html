How this Underdog payout calculator works

This free calculator applies Underdog's published base Standard and Flex payout tables to your entry amount. It can estimate a base payout and a visitor-supplied hit-rate scenario, but it cannot reproduce every entry-specific modifier or determine how Underdog will grade a live entry.

  • Base payouts: Standard entries currently range from 3.5x for two picks to 120x for eight picks. Flex sweep payouts range from 3.25x for three picks to 80x for eight picks, with smaller listed returns after eligible losses.
  • Ties, DNPs, and voids: a tied projection or qualifying DNP is generally voided. A still-valid entry pays based on its remaining active picks; an entry made invalid by a void is refunded.
  • Rescue: an eligible Higher selection may qualify when a player participates in the first half but does not return in the second. Rescue depends on entry, sport, projection, and promotion rules and remains at Underdog's discretion.
  • Limits: selection multipliers, correlations, promotions, contest type, voids, and state-specific formats can change the final amount. Underdog lists a 5,000x maximum multiplier for Standard and Flex entries, separate from the 120x eight-pick Standard base payout.

Always use the payout and grading shown by Underdog on the final entry screen. Propeller is an independent research tool. It is not affiliated with, endorsed by, or operated by Underdog.

Why the displayed Underdog payout can change

The tables above use 1.0x selections and Underdog's published base multipliers. Alternative projections, correlated projections, promotions, and contest formats can change the combined payout shown in the entry builder.

For Flex entries, lower-result amounts are up-to estimates. Which selection loses matters because its modifier no longer contributes to the payout. Treat this calculator as a base-table reference and confirm the actual multiplier in Underdog before submitting an entry.

Standard vs Flex: How the formats differ

Standard requires every active pick to be correct and uses the higher base sweep multiplier for a given pick count.

Flex uses a lower base sweep multiplier but can return a listed amount after one loss, or after two losses on six- through eight-pick entries. The actual return can depend on which selections lose and the modifiers attached to them.
