Questions

Before you use the board.

How often are MLB fantasy projections updated?

The public preview requests the current projection board when the page loads. Its status line shows the source update time and labels data older than 60 minutes as the latest available preview rather than live.

Do the projections include pitchers?

No. The current public fantasy projection board covers MLB hitters only.

Which scoring system is used?

The current projections use DraftKings MLB fantasy scoring. Always confirm the scoring rules for the contest or league where you plan to use the estimates.

Why can a floor or ceiling miss?

Because the range is an estimate derived from historical variation, not a hard boundary. Baseball outcomes, roles, weather, injuries, and scoring events can fall outside the modeled range.
